## Introduction
This homework is aims to simulate a signal $m(t)$ do base-band modulation.
- The signal $m(t)$ is defined as:
  $m(t)=[\cos(32\pi t) - \sin(32\pi t)]+[-\cos(36\pi t) - \sin(36\pi t)] + [\cos(44\pi t) + \sin(44\pi t)] + [-\cos(48\pi t) - \sin(48\pi t)]$
